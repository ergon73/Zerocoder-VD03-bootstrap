# Bootstrap Homework - Адаптивная веб-страница

## Описание проекта

Домашнее задание по курсу веб-разработки: создание адаптивной веб-страницы с использованием Bootstrap 5.3.3. Проект демонстрирует использование основных компонентов Bootstrap для создания современного одностраничного сайта.

## Структура страницы

- **Шапка**: навигационная панель (Navbar) с адаптивным меню
- **Слайдер**: карусель (Carousel) с 3 изображениями и подписями
- **Карточки**: 3 адаптивные карточки (Cards) с информацией об услугах
- **Аккордеон**: FAQ секция (Accordion) с 3 разделами
- **Футер**: контактная информация и ссылки на социальные сети

## Технологии

- **HTML5** - семантическая разметка
- **Bootstrap 5.3.3** - CSS-фреймворк для адаптивного дизайна
- **Адаптивный дизайн** - Mobile-first подход

## Особенности

- ✅ Полностью адаптивная верстка (mobile, tablet, desktop)
- ✅ Использование только Bootstrap классов (минимальный CSS для плавной прокрутки)
- ✅ Все компоненты интерактивны и функциональны
- ✅ Рабочая навигация с якорными ссылками и плавной прокруткой
- ✅ Семантическая HTML5 разметка
- ✅ Доступность (aria-атрибуты, alt-тексты)
- ✅ Стоковые изображения из Unsplash для карусели и карточек

## Как запустить

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/ergon73/Zerocoder-VD03-bootstrap.git
   ```

2. Откройте файл `index.html` в браузере

3. Или используйте локальный сервер:

   ```bash
   # Python
   python -m http.server 8000
   # Затем откройте http://localhost:8000/index.html

   # Node.js (http-server)
   npx http-server
   ```

## Адаптивность

Страница корректно отображается на всех размерах экранов:

- **Мобильные устройства** (< 768px): 1 колонка для карточек, свернутое меню
- **Планшеты** (≥ 768px): 2 колонки для карточек
- **Десктоп** (≥ 992px): 3 колонки для карточек, развернутое меню

## Компоненты Bootstrap

- **Navbar** - навигационная панель с toggle-меню и рабочими якорными ссылками
- **Carousel** - карусель с индикаторами, контролами и 3 слайдами
- **Cards** - 3 карточки в адаптивной сетке (col-12 col-md-6 col-lg-4)
- **Accordion** - аккордеон с 3 разворачивающимися секциями FAQ
- **Grid System** - адаптивная сетка Bootstrap
- **Utility Classes** - утилитарные классы для отступов и типографики

## Навигация

Навигационное меню включает рабочие ссылки:

- **Главная** → прокрутка к карусели (#home)
- **О нас** → прокрутка к карусели (#home)
- **Услуги** → прокрутка к секции карточек (#services)
- **Портфолио** → прокрутка к аккордеону (#portfolio)
- **Контакты** → прокрутка к футеру (#contacts)
